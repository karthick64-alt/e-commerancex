## üöÄ Getting Started in 3 Steps

### Step 1: Add Images
1. Create an account on [Unsplash](https://unsplash.com) or [Pexels](https://pexels.com)
2. Download images for:
   - Banners (3 images)
   - Categories (6 images)
   - Products (8+ images)
   - Collections, brands, users
3. Place them in `assets/images/` folder
4. See `IMAGE_SETUP.md` for detailed requirements

### Step 2: Open the Website
1. Simply open `index.html` in your web browser
2. Or use a local server for better experience:
   ```bash
   # Python
   python -m http.server 8000
   
   # Then visit: http://localhost:8000
   ```

### Step 3: Customize
1. Edit `assets/css/style.css` to change colors
2. Edit `assets/js/app.js` to modify products
3. Customize HTML files as needed
